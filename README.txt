В репозитории находятся задания по предметам Численные методы, Математическое программирование и Нейросети.

Приведены реализации:

~ Симплекс-метода
~ метода покоординатного спуска
~ метода Флетчера-Ривза
~ метода наискорейшего спуска
~ градиентного спуска с дроблением шага

А также пример обучения нейронной сети для задачи регрессии.
